# Consumer Safety Checklist for AI Skincare Tools

## Purpose

This checklist is designed to help evaluate whether an AI-supported skincare tool is responsibly positioned for non-clinical consumer decision-support.

It focuses on consumer safety, transparency, inclusion, and responsible communication.

## 1. Non-Clinical Positioning

- [ ] Does the tool clearly state that it does not provide medical diagnosis?
- [ ] Does it avoid claiming to treat or cure skin conditions?
- [ ] Does it encourage users to seek professional advice when needed?
- [ ] Does it use clear disclaimers?
- [ ] Does it position itself as education/support rather than medical authority?

## 2. Skin Tone Inclusion

- [ ] Does the tool consider melanin-rich skin?
- [ ] Does it explain whether diverse skin tones were considered in development?
- [ ] Does it avoid one-size-fits-all skincare assumptions?
- [ ] Does it recognise that some skin concerns may appear differently across skin tones?
- [ ] Does it communicate limitations clearly?

## 3. Product Recommendation Safety

- [ ] Does the tool avoid aggressive product recommendations?
- [ ] Does it warn users about overusing strong actives?
- [ ] Does it encourage patch testing where appropriate?
- [ ] Does it avoid unsafe ingredient combinations?
- [ ] Does it explain that individual skin reactions may vary?

## 4. Data and AI Transparency

- [ ] Does the tool explain that AI outputs may not always be accurate?
- [ ] Does it mention that lighting, image quality, and skin tone may affect results?
- [ ] Does it avoid presenting AI predictions as final conclusions?
- [ ] Does it provide understandable explanations for users?
- [ ] Does it allow room for human judgement?

## 5. Consumer Education

- [ ] Does the tool help users understand general skincare concepts?
- [ ] Does it explain ingredients in simple language?
- [ ] Does it guide users toward safer routine-building?
- [ ] Does it help users recognise when professional advice may be needed?
- [ ] Does it avoid fear-based messaging?

## 6. Ethical Communication

- [ ] Does the tool avoid exaggerated promises?
- [ ] Does it avoid shame-based beauty language?
- [ ] Does it avoid misleading before-and-after claims?
- [ ] Does it communicate respectfully with diverse users?
- [ ] Does it prioritise trust, clarity, and safety?

## Summary

A responsible AI skincare tool should educate, support, and guide consumers without replacing qualified professionals.

For melanin-rich consumers, safety and inclusion must be built into the design from the beginning.
